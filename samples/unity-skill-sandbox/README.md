# Unity Skill Sandbox

Minimal Unity-like sandbox for testing skills and deterministic scripts.

This is not a full game project. It exists to provide stable folders for architecture review, prefab planning, UI Toolkit planning, and script smoke tests.

Expected folders:

- `Assets/Game/Core`
- `Assets/Game/UI`
- `Assets/Game/Prefabs`
- `Assets/Game/ScriptableObjects`
- `Assets/Game/Tests/EditMode`
- `Assets/Game/Tests/PlayMode`
- `Packages/manifest.json`

Do not create Unity `.meta` files manually in this sandbox.
